# InsightU Telegram Bot

Telegram MVP for candidate intake.

## Features

- consent and privacy confirmation
- language switching: `ru`, `kk`, `en`
- step-by-step application flow
- saved progress
- back button
- essay and case response
- file, video and voice intake
- follow-up questions from backend
- final handoff to backend scoring pipeline

## Run

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
python main.py
```
